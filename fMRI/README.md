#fMRI projects


This directory contains all fMRI-related neuroimaging projects under the spectralFlow-DB framework


Structure:
- project_pilot/data/raw		: BIDS-compliant raw data (read-only)
- project_pilot/data/derivatives	: preprocessed outputs (fMRIPrep, etc.)
- project_pilot/code			: Analysis and preprocessing scripts
- project_pilot/results			: Final figures and tables

All analyses are executed in Ubuntu (WSL distribution) for reproducibility.
